## 说明

JQuery TMP，是基于[JQuery](https://jquery.com)框架与[BootStrap](https://v5.bootcss.com/docs/getting-started/introduction/)UI库制作的原生响应式模板。方便响应式官网等原生项目的快速搭建，节省开发周期，增加开发效率。
